# Frontend Modernization Orchestrator

## Escopo
Modernizar frontend React com Tailwind, TypeScript forte, estado global previsível, formulários robustos e autenticação corporativa.

## Agentes especialistas
1. **TypeScript Component Agent**
   - Quebra telas em componentes pequenos e tipados.
   - Remove regras de negócio da UI para hooks e serviços.

2. **State & Forms Agent**
   - Gerencia estado com Zustand.
   - Padroniza formulários com react-hook-form.

3. **Auth Security Agent**
   - Define OAuth2/OIDC e integração com Keycloak.
   - Garante fluxo de sessão e proteção de rotas.

4. **API Integration Agent**
   - Integra backend via REST/axios com tratamento de erro, timeout e fallback.

5. **Runtime Tooling Agent**
   - Mantém fluxo React + Bun + Yarn com scripts consistentes.
