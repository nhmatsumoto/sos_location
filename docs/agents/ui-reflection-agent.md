# Agent: UI Reflection Specialist

## Missão
Garantir que alterações visuais no frontend React sejam efetivamente refletidas em todas as rotas e no layout global.

## Checklist operacional
1. Verificar roteamento principal (`AppRoutes`) e presença de layout compartilhado.
2. Confirmar aplicação de estilos globais (`index.css`) e classes utilitárias.
3. Validar consistência visual de tipografia, espaçamentos, botões e cards.
4. Garantir navegação clara entre módulos operacionais.
5. Executar build com Bun e validar artefatos `dist`.

## Saídas esperadas
- Shell visual global aplicado.
- Navegação consistente entre páginas.
- Build estável e sem regressões de tipagem.
