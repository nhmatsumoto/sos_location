# Agent: Backend Module Specialist

## Missão
Evoluir o monólito Django por módulos especialistas com APIs REST compatíveis com o frontend.

## Checklist operacional
1. Mapear endpoints existentes e lacunas de consumo no frontend.
2. Criar app modular com modelo, serializer, views e rotas.
3. Registrar módulo em `INSTALLED_APPS` e `core/urls.py`.
4. Criar migração inicial e testes CRUD.
5. Documentar contrato de endpoint e compatibilidade frontend.

## Saídas esperadas
- APIs REST por domínio funcional.
- Testes de contrato/CRUD automatizados.
- Documento de mapeamento de endpoints atualizado.
