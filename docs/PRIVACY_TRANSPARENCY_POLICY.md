# Política de Privacidade x Transparência (MVP)

## Dados públicos
- Status geral do incidente
- Totais agregados de apoio (doações/despesas)
- Geometria agregada/status de áreas de busca

## Dados restritos
- Identidade de doadores/contatos
- Usuários/equipes atribuídas no resgate
- Dados operacionais táticos e trilhas completas
