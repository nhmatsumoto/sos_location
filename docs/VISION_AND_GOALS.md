# Visão Geral e Objetivo

O MG Location é uma plataforma para apoio operacional em resposta a desastres.

## Objetivos
- Melhorar tempo de resposta.
- Apoiar coordenação com dados geoespaciais.
- Unificar dados de clima, alertas, transparência e satélite.
- Viabilizar cadastros de campo com persistência.
