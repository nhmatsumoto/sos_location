# React Hook Form Patterns

## Workflow
1. Definir tipo do payload do formulário.
2. Inicializar `useForm` com default values.
3. Aplicar validações mínimas por campo crítico.
4. Integrar submit com hook de domínio.
5. Reset/cancel conforme fluxo de edição/criação.
