# TypeScript Componentization

## Workflow
1. Identificar blocos visuais repetidos e extrair componentes.
2. Definir `types/` central para contratos de dados.
3. Mover lógica de IO e regras para `hooks/` e `services/`.
4. Garantir props mínimas e tipagem estrita.
5. Validar com lint/build.
